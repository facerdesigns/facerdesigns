FACER DESIGNS - IMAGE FOLDER INSTRUCTIONS
==========================================

This folder contains images for your portfolio website.

FOLDER STRUCTURE:
-----------------
/images/
  └── /gallery/     <- Put your portfolio project photos here

HOW TO ADD YOUR PHOTOS:
------------------------

1. PREPARE YOUR IMAGES:
   - Download photos from your Facebook portfolio
   - Recommended size: 800x800 pixels (square format works best)
   - Supported formats: JPG, PNG
   - Keep file sizes under 1MB for fast loading

2. NAME YOUR IMAGES:
   - project-1.jpg
   - project-2.jpg
   - project-3.jpg
   - project-4.jpg
   - project-5.jpg
   - project-6.jpg
   - project-7.jpg
   - project-8.jpg
   - project-9.jpg

3. PLACE IMAGES IN GALLERY FOLDER:
   - Copy your photos to: /home/facer/images/gallery/
   - Or in Windows: \\wsl.localhost\Ubuntu-24.04\home\facer\images\gallery

4. UPDATE IMAGE DESCRIPTIONS (Optional):
   - Open index.html
   - Find the gallery section
   - Update the text inside each gallery-overlay:
     * <h3> = Title of the project
     * <p> = Brief description

5. ADD MORE GALLERY ITEMS:
   - Copy an existing gallery-item div
   - Change the image src to your new image name
   - Update the title and description
   - The grid will automatically adjust

TIPS:
-----
- Use descriptive filenames for better organization
- Keep images consistent in size for best appearance
- You can have as many gallery items as you want
- The gallery displays 3 columns on desktop, 1 column on mobile

CURRENT GALLERY ITEMS:
----------------------
The website currently expects 9 images named project-1.jpg through project-9.jpg
You can add more or use fewer - just update the HTML accordingly.
